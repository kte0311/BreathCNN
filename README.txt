# BreathCNN
CNN 들숨,날숨 분류 모델

# 데이터 셋 설명
본 과제에서 활용되는 데이터셋은 .wav 형식의 인간 호흡 소리 오디오 파일로 구성되어 있으며, train/ 및 test/ 폴더에 각각 학습용과 테스트용 파일이 저장되어 있다. 각 오디오 파일에 대한 메타데이터는 train.csv 및 test.csv 파일로 제공되어, 모델 학습 및 평가에 활용된다.

사용 데이터 파일 
1-1) train/ (폴더)
모델 학습에 사용되는 .wav 형식의 오디오 파일들이 담겨 있으며, 각 파일은 한 사람의 호흡 소리(들숨 또는 날숨)를 담고 있다.
1-2) test/ (폴더)
모델 성능 평가에 사용되는 .wav 형식의 오디오 파일들이 담겨 있고, 파일들에 대해 들숨/날숨을 예측해야 한다.
1-3) train.csv
train/ 폴더의 오디오 파일들의 ID와 이에 대한 정답 레이블(I or E)을 포함하고 있는 csv파일로, 이 파일을 사용하여 모델을 학습시킬 수 있다.
- ID 컬럼
● file_name: 오디오 파일의 고유 식별자(파일 이름). train/ 폴더의 .wav 파일과 일치
● label: 해당 오디오 파일의 정답 레이블
- Target 컬럼
● I: 들숨 (Inhale)
● E: 날숨 (Exhale)
1-4) test.csv
우리가 예측해야 할 test/ 폴더 안의 오디오 파일들의 ID 목록이 들어있다.
